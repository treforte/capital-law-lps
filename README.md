# Capital Law Firm — PPC Landing Pages

Static HTML/CSS/JS landing pages for The Capital Law Firm (Personal Injury, Los Angeles / California).

One folder per ad group at the repo root. No build step — deployed via GitHub Pages.

## Live pages

| Ad group | URL |
|----------|-----|
| Brand | https://treforte.github.io/capital-law-lps/brand/ |
| Premises Liability | https://treforte.github.io/capital-law-lps/premises-liability/ |

## Adding a new landing page

1. Create a new folder at the repo root (e.g. `/car-accidents/`)
2. Add `index.html` (and an `assets/` subfolder for images)
3. Commit and push — GitHub Pages deploys automatically
